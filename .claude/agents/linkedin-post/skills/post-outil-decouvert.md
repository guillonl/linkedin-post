# Compétence : Post découverte d'outil/feature IA

## Objectif
Faire découvrir un outil, une feature ou une technique peu connue.

## Structure recommandée

1. **Hook découverte (1-2 lignes)**
   - Référence au screenshot/visuel de l'outil
   - Enthousiasme naturel
   - Exemple : "J'ai découvert un truc cool"
   - Exemple : "Vous connaissiez cette feature dans [outil] ?"

2. **Pourquoi c'est utile (2 lignes)**
   - Use case concret
   - Pour qui c'est pertinent

3. **Comment ça marche (3-5 lignes)**
   - Étapes simples
   - Où trouver la feature
   - Settings importants
   - Peut être en points numérotés si + de 3 étapes

4. **Mon usage perso (2 lignes)**
   - Dans quel contexte tu l'utilises
   - Exemple concret de ton workflow

5. **Limite/précision (1 ligne - optionnel)**
   - Ce que ça ne fait PAS (gérer les attentes)

6. **CTA découverte (1 ligne)**
   - "Vous utilisez quoi d'autre pour ça ?"
   - "D'autres features cachées à partager ?"

## Ton
- Enthousiasme authentique
- Pédagogique
- "Regardez ce que j'ai trouvé !"

## Formulations
- "Petit game changer..."
- "Feature méconnue qui mérite le détour"
- "Ça faisait longtemps que je cherchais ça"

## Longueur cible
140-200 mots
