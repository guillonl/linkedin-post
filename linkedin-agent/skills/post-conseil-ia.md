# Compétence : Post conseil IA actionnable

## Objectif
Partager un conseil pratique sur l'utilisation de l'IA dans le workflow design.

## Structure recommandée

1. **Hook visuel (1-2 lignes)**
   - Référence directe à l'image
   - Exemple : "Sur l'image, vous voyez comment j'utilise Claude pour..."
   - Ou : "Ce workflow que vous voyez m'a fait gagner 3h cette semaine."

2. **Contexte rapide (2 lignes)**
   - Pourquoi ce besoin
   - Avant/après ou problème initial

3. **Le conseil concret (3-5 lignes)**
   - Étapes claires et actionnables
   - Outils mentionnés
   - Paramètres spécifiques si pertinent
   - Pièges à éviter

4. **Bénéfice concret (1-2 lignes)**
   - Temps gagné / qualité améliorée
   - Ce que ça change vraiment

5. **CTA pédagogique (1 ligne)**
   - Question : "Vous utilisez quel outil pour ça vous ?"
   - Invitation : "Testez et dites-moi si ça marche pour vous"
   - Ouverture : "D'autres astuces dans les commentaires ?"

## Formules à privilégier
- "Voilà comment je fais..."
- "J'ai galéré avec X, maintenant je fais Y"
- "Petit tip qui change tout..."
- "Si vous faites encore X à la main, regardez ça..."

## Formules à éviter
- "Le secret que personne ne vous dit"
- "Vous faites TOUT FAUX"
- "Cette méthode RÉVOLUTIONNAIRE"

## Ton
- Collègue bienveillant
- Enthousiaste mais réaliste
- Humble : "ça marche pour moi, à tester chez vous"

## Longueur cible
150-220 mots
